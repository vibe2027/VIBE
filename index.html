// Justice · Tribunal Communautaire

Une justice rendue
par la communauté.

Pas de modérateur anonyme caché derrière un écran. Tous les conflits sont arbitrés par un panel tournant de membres volontaires — vote secret, transparence totale.

// 01 · TRIBUNAL VERDICT
Coupable ? 7 jours en mode lecture seule, ou 20 $ pour le pardon.
Si le tribunal te juge coupable, ton accès devient passif : tu vois mais tu ne peux rien faire pendant 7 jours.
// 02 · PANEL TOURNANT
12 membres tirés au sort. Identités protégées. Vote secret.
Chaque panel est différent. Aucune clique ne peut contrôler le verdict. La majorité décide.
// 03 · BANNISSEMENT
3 verdicts coupables = exclusion définitive. Sans appel.
Les récidivistes ne reviennent pas. La communauté se protège elle-même.
